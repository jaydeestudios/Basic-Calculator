# 📱 Basic Calculator

A sleek, dark-themed calculator built with a focus on modern UI/UX principles and clean logic. 

## 🌟 Features
- **Modern Design:** Uses a deep-sea dark theme with neon blue accents.
- **Responsive Layout:** Built with **CSS Grid** to ensure it looks great on any screen size.
- **Clean Logic:** Handles basic arithmetic with precision and includes a "Delete" function for corrections.
- **Python Backend Concept:** Includes a `logic.py` script for potential expansion into advanced mathematical functions.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Modern Grid & Flexbox)
- **Logic:** JavaScript (ES6)
- **Backend Exploration:** Python 3.12
